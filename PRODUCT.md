# Product

## Register

product

## Users

นักเล่นหวยไทยและหวยลาวพัฒนา ที่ต้องการข้อมูลเชิงสถิติประกอบการตัดสินใจ ไม่ใช่ผู้เชี่ยวชาญ Data Science แต่เชื่อในข้อมูล ใช้บนมือถือและ desktop, มักเปิดก่อนวันออกรางวัล 1–3 วัน บรรยากาศการใช้งาน: โต๊ะ, บ้าน, ร้านกาแฟ ในเวลากลางคืนเป็นหลัก

## Product Purpose

LottoLens วิเคราะห์สถิติผลรางวัลสลากกินแบ่งรัฐบาลไทยและหวยลาวพัฒนา ด้วย Data Science และ AI ให้ผู้ใช้เห็น Pattern ความถี่, ตัวเลขร้อน-เย็น-overdue, แนวโน้ม, และ AI prediction เพื่อประกอบการตัดสินใจ ไม่ใช่เพื่อรับประกันผล ความสำเร็จคือผู้ใช้เชื่อมั่นในตัวเลขที่เลือก และกลับมาใช้ซ้ำทุกงวด

## Brand Personality

น่าเชื่อถือ, ฉลาด, โปร่งใส — เครื่องมือที่ดูจริงจังแต่เข้าถึงได้ ไม่หลอกว่าทำนายได้ 100% แต่แสดงข้อมูลอย่างมีสีสันและน่าสนใจ

## Anti-references

- ❌ เว็บหวยไทยทั่วไปที่รกด้วย banner โฆษณาและตัวเลขสีฉูดฉาด
- ❌ SaaS dashboard ที่เย็นชาและไร้บุคลิก (navy + white generic)
- ❌ Crypto site ที่ใช้ neon บน black ทุกอย่าง
- ❌ Hero-metric template: big number + small label แบบ SaaS cliché

## Design Principles

1. **ข้อมูลก่อนการตกแต่ง** — ทุก visual element ต้องช่วยให้อ่านตัวเลขได้เร็วขึ้น ไม่ใช่ทำให้สวยงามอย่างเดียว
2. **ความซื่อสัตย์ต่อผู้ใช้** — แสดง disclaimer ชัดเจน ไม่เกินจริง สถิติคือเครื่องมือไม่ใช่คำทำนาย
3. **Mode ที่ชัดเจน** — Thai / Lao mode ต้องรู้สึกแตกต่างกันอย่างชัดเจนแต่อยู่ใน design system เดียวกัน
4. **ใช้งานได้เร็ว** — ผู้ใช้ต้องการตัวเลขทันที navigation ต้องไม่มีขั้นตอนเยอะ
5. **เชื่อถือได้เหมือนเครื่องมือ** — UI ต้องรู้สึก solid และ consistent ไม่ใช่ experimental

## Accessibility & Inclusion

- ภาษาหลัก: ภาษาไทย (UI copy ทั้งหมด)
- WCAG AA เป็นเป้าหมาย
- รองรับ keyboard navigation และ screen reader ขั้นพื้นฐาน
- รองรับ prefers-reduced-motion สำหรับผู้ใช้ที่ไวต่อ animation
- Light/Dark mode toggle: ผู้ใช้เลือกได้เอง
